Conclusao de AstroTech GMW

    O projeto foi planejado devidamente e implementado dados tanto no planejamento quando no desenvolvimento e conclusao. Feito um planejamento onde o cliente requisitou 
1. Nome Completo
2. CPF
3. Contato (Telefone)
4. Contato Secudário (Telefone)
5. Endereço (CEP, Rua, Número, Complemento).
    Depois de analisado os requisitos foi feito a parte para desenvolver o codigo, onde ocorreu varias modificacoes tais como:
1.Implementacao de dados
2.melhor experiencia do cliente |(layout)
3.filtros para remover dados incorretos
4.Sepracao de numericos e strings
    O programa ficou clean e responsivo, com validacao basica de CPF e e-email, com sua barra de procura de clientes funcional armazenando as informacoes em um localStorage.
